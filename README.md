# react js & ts templates

## How ro use
```bash
# Step 0
# Clone repo
$ git clone git@github.com:guiboli/react-templates.git

# Step 1
# Copy templates
# js templates
$ cp -rf react-templates/templates/js <project_path>
# or ts templates
$ cp -rf react-templates/templates/ts <project_path>

# Step 2
# install deps and start project
# cd <project_path>
$ yarn
$ yarn start
